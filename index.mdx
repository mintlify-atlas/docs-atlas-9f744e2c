SM64 Coop DX Launcher (`coopdx-rs`) is a Rust-built graphical launcher for [Super Mario 64: Coop Deluxe](https://sm64coopdx.com) on Linux. It handles game path detection, ROM validation, mod management, DynOS packs, player profiles, and multiplayer network configuration — so you can spend less time configuring and more time playing.

## What the launcher does

<Steps>
  <Step title="Finds the game binary">
    The launcher resolves the SM64 Coop DX binary through a 5-tier priority chain: CLI argument → environment variable → per-profile override → `launcher.toml` → automatic directory search.
  </Step>
  <Step title="Validates the ROM">
    Before launching, it locates a valid Super Mario 64 (US) ROM by MD5 checksum (`20b854b239203baf6c961b850a4a51a2`) and copies it to the correct save path automatically.
  </Step>
  <Step title="Applies your configuration">
    Active mods, DynOS packs, network mode, and profile options are assembled into game CLI arguments (`--enable-mod`, `--server`, `--playername`, etc.) and passed directly to the binary.
  </Step>
  <Step title="Spawns the game cleanly">
    The game process is spawned with a sanitized environment (`env_clear()` + whitelist) to prevent library conflicts from the launcher or system environment leaking into the game.
  </Step>
</Steps>
